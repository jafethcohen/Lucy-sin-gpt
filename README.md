# Lucy — Voice Accessibility Assistant 🎙️

Lucy is a voice-controlled assistant designed to help people with physical disabilities use a computer through voice commands, without needing a mouse or keyboard.

## What it does

- Listens to voice commands in real time
- Executes system actions (open apps, navigate, control the OS)
- Designed with accessibility as the core principle

## Motivation

Many people with motor disabilities struggle to use conventional input devices. Lucy aims to bridge that gap using Python and speech recognition.
